# Parkinson's Disease Prediction using Machine Learning

This project predicts Parkinson’s disease using machine learning techniques.

## Technologies Used
- Python
- Flask
- Machine Learning
- Pandas, NumPy, Scikit-learn
- HTML, CSS

## Features
- User-friendly web interface
- Predicts Parkinson’s disease based on input parameters
- Trained ML model integrated with Flask backend

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/

## Output
The system displays whether the patient is likely to have Parkinson’s disease or not.
